Поддерживаемые операторы:

  • AND - пересечение (оба термина): python AND программирование
  • OR - объединение (любой термин): python OR java
  • NOT - исключение: python AND NOT javascript
  • Неявный AND - два термина подряд: машинное обучение

Примеры запросов:

  • python
  • python AND программирование
  • машинное AND обучение
  • веб OR мобильный
  • python AND NOT javascript
  • веб разработка OR программирование
